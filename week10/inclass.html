In-class Exercises

Part 1: Mapbox

  1. Log in to your Mapbox account and go to Studio.
  2. Create a new style based on Basic.
  3. Change the color of some polygons and lines.
  4. Change the font of some labels.
  5. Make buildings more transparent when zoomed out and less transparent when zoomed in.
  6. Experiment!
  7. Publish your style and add it to a Carto map as a basemap.

Part 2: Add a Mapbox layer

  1. Mapbox Basic doesn't contain state boundaries in the US. Let's fix this.
  2. Add a new layer by clicking + Layer in the top left.
  3. Click the button next to Source and select admin.
  4. Click Create Layer and style it.
  5. Note that the boundaries in the ocean are included, too. That's because the US boundary is included here—let's filter country boundaries out. Click the Select data tab with your admin layer selected. Make a filter for admin_level is any of with the value 4 (states).

Part 3: Add your own layer

  1. Mapbox Basic also doesn't contain the borough boundaries. Let's add those now.
  2. Download today's data and unzip it. The zip file nybb.zip contains the borough boundaries shapefile.
  3. Add a new layer to your style. Instead of using an available layer, click New tileset and upload nybb.zip.
  4. Once the upload is complete, select it and style the layer.
