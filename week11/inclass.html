In-class Exercises

Part 1: OSMDownloader

  1. Install the OSMDownloader QGIS plugin.
  2. Zoom to a small (part of a city or smaller) area on the map, using a base map or similar to find an area you want to map.
  3. Download data by going to Vector > OpenStreetMap > Download Data. In the dialog select Extent: From map canvas. Don't forget to browse for somewhere to put the data and make sure the filename ends with .osm.
  4. Open the data—drag and drop it into OSM, select all the layers, and press OK.

Part 2: Editing OSM

  1. Go to openstreetmap.org and create an account if you don't already have one.
  2. Log in and click the Edit button. You should be prompted to do the walkthrough. Take 10 minutes to follow the walkthrough.
  3. Once you're done with the walkthrough, move the map to an area in the world you're familiar with. This could be the neighborhood you live in now, where you grew up, etc. Find something you can map and give it a shot. I find many areas, especially rural ones, have buildings that could be traced. Anything that exists in the real world is valid to add to OpenStreetMap.

Part 3: Time Manager

  1. Download this data.
  2. Open the earthquakes shapefile in QGIS.
  3. Install the TimeManager QGIS plugin.
  4. Use TimeManager to animate the earthquakes using the time column with a time frame of one day.
