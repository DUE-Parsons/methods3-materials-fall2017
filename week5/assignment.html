Assignment 5

This week we talked about another way to unlock more geographic data (geocoding) and a few other ways to represent density using QGIS (heat maps and hex bins). Use this assignment to get familiar with at least one of these and make another map in QGIS before we move to putting maps online next week.

Please note that this assignment is shorter than other assignments in order to give you more time to work on the Final Project Proposal.

Geocoding, heat maps, or hex bins

Geocoding, heat maps, or hex bins. Pick the one that's most interesting to you and improve one of your previous maps or make a new one. If you change a previous map, compare it with the new one in your response.

When geocoding you will want a CSV with addresses. This usually comes up naturally, and it can be hard to find a suitable file on purpose. Here are some data sources you might use:

  • Self Hauler Registrants (ignore the latitude and longitude columns)
  • HIV Testing Locations

For heat maps and hex bins you will need dense point data. Here are some data sources you might find interesting that you're welcome to use:

  • Inside Airbnb for any city. (Use the file named listings.csv for the city you choose.)
  • NYC Street Tree Census
  • USGS Earthquakes (download one of the GeoJSON files on the right)

Submitting Your Work

Use the QGIS Print Composer to create one PDF of your map. Make sure to add map accessories such as legends, titles, and scale bars as appropriate. Refer to your sources as appropriate.

Finally, include a short summary of what you did for this assignment. Explain what you created and how you did so. What did you have problems with? What did you enjoy? What would you like to know how to do next?
