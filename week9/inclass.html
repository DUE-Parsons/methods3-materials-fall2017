In-class Exercises

Part 1: HTML

  1. Download today's data.
  2. Upload the zipped shapefile to your Carto account (airbnb-listings.zip). This is slightly different from the other Airbnb files we have used in the past.
  3. Make a map using the data.
  4. Let's add a link back to the source (insideairbnb.com). Add a legend, edit the HTML for it, and add this at the bottom of the legend (you can copy and paste it): 
    <p>
    Data from <a href="http://insideairbnb.com/">Inside Airbnb</a>
</p>
  5. Let's link to each listing's page on Airbnb. Add a popup for the layer and add the room_url field to it. Then edit the HTML and find {{room_url}} in it. Replace {{room_url}} with this: 
    <a href="{{room_url}}">View on airbnb</a>
  6. If you want the links to open in their own tab, use this instead: 
    <a href="{{room_url}}" target="_blank">View on airbnb</a>

Part 2: CSS

  1. Let's make the link in the legend blend in better with the rest of it. We'll add top margin to the link to make space between it and the legend, then make the font a bit smaller. In the legend, update our current link: 
    <p>
    Data from <a href="http://insideairbnb.com/">Inside Airbnb</a>
</p>
    with this: 
    <p style="margin-top: 20px; font-size: 10px;">
    Data from <a href="http://insideairbnb.com/">Inside Airbnb</a>
</p>
  2. You may want to change more font styles, too. Try adding a style to the link, ending up with this: 
    <p style="margin-top: 20px; font-size: 10px;">
    Data from <a href="http://insideairbnb.com/" style="color: #820575; font-weight: bold;">Inside Airbnb</a>
</p>
    Feel free to try a different color.
